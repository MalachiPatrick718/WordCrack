# MindShift Landing (Static)

Static marketing site for MindShift (no backend).

## Local dev

```bash
cd "mindshift-landing"
npm install
npm run dev
```

## Static export build

```bash
cd "mindshift-landing"
npm install
npm run build
```

The static site is exported to `mindshift-landing/out/`.

## Deploy

- Vercel: import this folder as a project.
- Any static host: upload the `out/` directory.
