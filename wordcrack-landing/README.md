# WordCrack Landing (Static)

Static marketing site for WordCrack (no backend).

## Local dev

```bash
cd "wordcrack-landing"
npm install
npm run dev
```

## Static export build

```bash
cd "wordcrack-landing"
npm install
npm run build
```

The static site is exported to `wordcrack-landing/out/`.

## Deploy

- Vercel: import this folder as a project.
- Any static host: upload the `out/` directory.
